# localtransfer-py

A Python-based local file tranferer with security and simplicity in mind.

## Basic usage:

```
py app.py --path C:/your/path
```

```
py app.py --autopassword --length 8
```

```
py app.py --password 12345678
```

```
py app.py (arguments) --lifespan 50000000000
```

*Note*: `--lifespan is in seconds`
